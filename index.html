HDFS Cluster sizing 
          Formula: H = C * R * S/(1+i) * (1 + r)
          C - Compression Ratio
          R - Replication factor
          S - Initial data size
          i - Intermediate data factor, typically 1/3 or 1/4 (temporary space for mapreduce)
          r - Reserved ratio for the underlying OS and filesystem
